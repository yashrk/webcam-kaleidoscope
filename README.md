# webcam-kaleidoscope

## Keyboard controls

| Key            | Action                 |
|----------------|------------------------|
| `r`            | Start/stop rotation    |
| `m`            | Change mesh            |
| `←`, `→`       | Change fragment shader |
| `↑`, `↓`       | Change vertex shader   |
| `PgUp`, `PgDn` | Change vertex shader   |
| `ESC`          | Exit                   |
