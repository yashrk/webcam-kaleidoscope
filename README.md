# webcam-kaleidoscope
## Standard controls
### Keyboard controls

| Key            | Action                 |
|----------------|------------------------|
| `r`            | Start/stop rotation    |
| `m`            | Change mesh            |
| `←`, `→`       | Change fragment shader |
| `↑`, `↓`       | Change vertex shader   |
| `PgUp`, `PgDn` | Change camera height   |
| `ESC`          | Exit                   |
| `9`            | Slow down rotation     |
| `9`            | Speed up rotation      |

### Mouse wheel
Change camera height

## Local configuration
To redefine config params put `local.json` file to `config` folder.
